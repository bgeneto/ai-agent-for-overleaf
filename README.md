- [Edge Add-on](https://microsoftedge.microsoft.com/addons/detail/overleaf-copilot/dgbgphmgphkibogcjhjhdmkjphejcead)

## Privacy Policy

This extension **does not** collect or transmit any user data to its own servers.
- **No Third-Party Tracking**: We do not use analytics or tracking cookies.
- **Direct Communication**: All communication is performed directly between your browser and the OpenAI API (or your configured compatible provider) using the API Key you provide.
- **Local Storage**: Your API Key and preferences are stored locally in your browser (`chrome.storage.local`) and are encrypted (obfuscated) for additional security.
- **No Fallback**: Unlike previous versions, this extension does not use any hosted fallback services. If no API Key is provided, the extension will not function.
